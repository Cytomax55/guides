# Telegram Notification Bot

You can use the [Harmony Analytics Bot](https://t.me/HarmonyAnalyticsBot) on Telegram to get notification on changes to wallets or to validators. Hit up the telegram bot to get started.

```text
# To pull up the menu send
/menu

# To set your favorite wallets add, mutliples separate with a comma no spaces
/setaddress <one wallet>

# To set your validator address
/setfav <one validator wallet>
```
